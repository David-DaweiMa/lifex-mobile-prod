# Coly 执行清单（任务分解 / 负责人 / 时间线）

## 时间线（8 周路线图）
- Week 1：准备与设计评审
- Week 2：抓取与提醒框架MVP
- Week 3：降价命中/First Table监测
- Week 4：周报卡片与频控
- Week 5：历史价图/替代推荐
- Week 6：灰度与验证
- Week 7：AI 推荐基础（向量索引、偏好融合）
- Week 8：客户分群与支付集成试点

## 角色
- 产品（P）：范围、验收、文案
- 工程（E）：数据抓取、后端、客户端
- 数据（D）：SKU映射、质量监控
- 设计（U）：UI/UX

## 任务清单
- 1) 评审与对齐（W1） [P/U/E]
  - 评审 PRD、DB 提案、数据管线、UI 规范
  - 输出：冻结范围与验收
- 2) 抓取框架 MVP（W2） [E]
  - Cheerio 抽取器规范、限速/退避策略、失败回退流程（文档实现）
  - 定时调度与分批（Cron+队列）设计
- 3) 数据域与表（W2） [E/D]
  - 确认 deals/slots/supermarket_* 字段与索引
  - 软过期与保留策略
- 4) 降价命中与阈值（W3） [E/D]
  - 规则：降幅%、降价$；命中生成通知（合并策略文档）
- 5) First Table 监测（W3） [E]
  - Top 餐厅映射；日更2次；命中心愿时段→通知
- 6) 周报卡片与频控（W4） [U/E]
  - 卡片模板、排序、折叠；每周1次+实时降频
- 7) 历史价图与替代推荐（W5） [D/E]
  - 采样与存储；同城同类性价比规则
- 8) 质量与监控（W5） [D/E]
  - 覆盖率、失败率、命中率面板；报警与降级
- 9) 灰度验证（W6） [P/U/E]
  - 指标：打开/点击/留存/节省；文案A/B

## 里程碑验收
- M1（W2）：当周促销抓取+定时调度文档完成
- M2（W3）：降价命中通知 + First Table 监测文档完成
- M3（W4）：周报卡片规范与频控策略就绪
- M4（W6）：灰度数据达标（打开≥35%，点击≥12%）
- M5（W7）：语义检索+偏好融合离线评测通过（NDCG@10≥0.6）
- M6（W8）：支付与分群试点上线（无阻断Bug）
